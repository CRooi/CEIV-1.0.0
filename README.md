# CEIV (Chinese Earthquake Information Viewer)
🌐 一个开源的中国地震信息可视化项目。

基于HTML、JavaScript、Mapbox GL JS。

## ✅功能
- ℹ️ 中国地震速报信息整合
- 🚨 中国地震预警信息整合
  - ⭕️ 横、纵波可视化显示
  - 1️⃣ 自定位置横波抵达倒数、烈度粗估
- ℹ️ 重庆北碚测站信息整合 (https://wolfx.jp/home/rpishake.html)

## ✅演示站点
https://app.projectbs.cn/ceiv/index-formal.html

## ⚠️开源协议
本项目仅可**独立使用**，请勿用于网页（软件）镶嵌等场景。
